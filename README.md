## Project Overview

This project analyzes Python from an open-source perspective and demonstrates Linux skills using five shell scripts. It highlights both theoretical understanding and practical implementation.

## Scripts Description

* Script 1: Displays system information like kernel, user, uptime, and OS
* Script 2: Checks if Python is installed and shows package details
* Script 3: Audits directories, permissions, and disk usage
* Script 4: Analyzes log files and counts keyword occurrences
* Script 5: Generates a personalized open-source manifesto

## How to Run

1. Give permission:
   chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh

2. Run scripts:
   ./script1.sh
   ./script2.sh
   ./script3.sh
   ./script4.sh /var/log/syslog error
   ./script5.sh

## Requirements

Linux OS, Bash shell, and basic commands like uname, dpkg, grep.

## Conclusion

This project demonstrates open-source philosophy and practical Linux scripting skills using Python as the core example.
